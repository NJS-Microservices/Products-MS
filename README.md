# Products Microservice

### Development
1. Clone the repository
2. Install the dependencies
3. Create the `.env` file with the environments variables (Example in `.env-template` file)
4. Execute Prisma migrations with the command `yarn prisma migrate dev`
5. Execute the command `yarn start:dev` to start the application in development mode
